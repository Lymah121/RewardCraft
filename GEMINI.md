# Directory Overview

This directory contains research materials for the development of a game-based educational tool designed to teach Reinforcement Learning (RL) to HIGH SCHOOL students (ages 14-18). The project has evolved to target older students who can engage with more complex reward function design and become true AI system designers. The other files are supporting academic papers and resources that form the foundational literature for this work.

# Usage & Project Goals

This directory is a workspace for a research project that aims to:

1.  **Develop a new game-based tool** for teaching Reinforcement Learning to middle schoolers.
2.  **Innovate** by combining deep, parametric interactivity with compelling, embodied visualization.
3.  **Integrate ethics** directly into the gameplay by allowing students to explore the consequences of their reward function designs.
4.  **Contribute to academic research** by addressing gaps in longitudinal studies, embedded assessment, and the transfer of learning in AI education.

The project is grounded in established learning science and aims to produce a pedagogically rigorous and engaging tool.
