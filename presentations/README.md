# RewardCraft Presentations

This folder contains presentation materials for the RewardCraft project.

## Project Overview

**RewardCraft** is an educational AI tool that teaches Reinforcement Learning (RL) concepts to high school students through an interactive tower defense game. Students design reward functions and watch an AI agent learn from their designs in real-time.

## Presentation Materials

### Phase 1: Core Implementation
- Q-Learning tower defense game
- Real-time training visualization
- WebSocket-based live updates
- Q-Table heatmap visualization

### Phase 2: Enhanced Learning Experience
- Learning curve visualization (Chart.js)
- Configurable training speed (0.5x - 10x)
- Hyperparameter controls (α, γ, ε)
- Reward breakdown panel
- Save/Load trained agents
- Educational tooltips

## Key Talking Points

### Educational Value
1. **Constructionist Learning**: Students are AI trainers, not passive players
2. **Embedded Ethics**: Reward function design reveals AI alignment challenges
3. **Specification Gaming**: Students experience unintended consequences firsthand
4. **Progressive Complexity**: From basic toggles to full hyperparameter control

### Technical Innovation
1. **Real-time Visualization**: Watch the AI learn step-by-step
2. **Transparent AI**: Q-values and decision-making are visible
3. **Immediate Feedback**: See how reward changes affect behavior
4. **Experiment & Compare**: Save different reward configurations

### Target Audience
- High school students (ages 14-18)
- K-12 educators
- AI education researchers

## Demo Script

1. **Introduction** (2 min)
   - What is Reinforcement Learning?
   - Why game-based learning?

2. **Interface Overview** (3 min)
   - Game canvas (tower defense)
   - Q-Table heatmap
   - Reward designer

3. **Live Demo** (5 min)
   - Design a reward function
   - Start training at slow speed (1x)
   - Watch AI make decisions
   - Observe learning curve

4. **Experiment** (5 min)
   - Modify rewards (e.g., high penalty for tower building)
   - Observe different behavior
   - Discuss specification gaming

5. **Advanced Features** (3 min)
   - Hyperparameter tuning
   - Save/compare agents
   - Q-Learning formula explanation
